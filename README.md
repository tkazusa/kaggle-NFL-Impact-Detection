# kaggle-NFL-Impact-Detection

## Preparation
```
> pip install kaggle
> mkdir ~/.kaggle
> cp .kaggle.json ~/.kaggle/
> chmod 600 ~/.kaggle/kaggle.json
> kaggle competitions download -c nfl-impact-detection
> makdir data/input
> unzip nfl-impact-detection.zip -d ./data
```
